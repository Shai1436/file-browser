# file-browser
It is a MEAN file browser.weas
How to use?
  1. Clone the repository.
  2. cd client
  3. npm install && bower install
  4. grunt serve
  5. cd ../server
  6. npm install
  7. npm start
  8. Go to localhost:9000
  
 Enter the abolute path of folder you want to browse, in the input box. Click Browse.
